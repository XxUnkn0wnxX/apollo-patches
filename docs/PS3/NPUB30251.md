# Dungeon Defenders (PSN) 

`NPUB30251.savepatch`

### 1. Decompress Save (Required)

Target File: `PAYLOAD`

```
Decompress PAYLOAD
```

### 2. Compress Save (Required)

Target File: `PAYLOAD`

```
Compress PAYLOAD,-w 15
```

### 3. Update CRC32Big for PAYLOAD (Required)

Target File: `PAYLOAD`

```
set pointer:eof+1
set range:0x000014,pointer
set [hash]:CRC32Big
write at 0x000010:[hash]
```
